Overview

Demo 结构

v1

均匀 baseline 时序，所有内容字统一设为 170 ms。

v2

保持内容字 170 ms，只加入关键边界停顿 260 ms。

v3

保留关键停顿，并把重点局部内容 token 拉长到 300 ms。

Spontaneous

不提供任何显式 text duration，由模型自行决定节奏。

Reading Guide

What changes across v1 / v2 / v3?

`v1` is a uniform baseline. `v2` keeps the same content duration but adds a critical pause. `v3` keeps that pause and further stretches the key local tokens.

What is spontaneous mode?

No explicit target-side duration is provided. MAGIC-TTS predicts internal duration and pause timing on its own.

Included scenes

Navigation turn, kids reading, accessibility code, and station arrival, all spoken with the same default voice.
