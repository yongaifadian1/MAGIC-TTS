# Default English Prompt Assets

This directory contains the built-in English prompt used by the release layer
when `--language en` is selected and no custom prompt is provided.

Files:

- `prompt.mp3`: default English prompt audio
- `prompt.txt`: matching English prompt transcript
- `prompt_track.json`: prompt-side duration track reused from existing MFA sidecar
- `prompt_alignment_raw.json`: stored source alignment payload for the prompt
- `prompt_alignment_debug.json`: metadata describing how the prompt track was built
