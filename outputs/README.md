# Release Demo Assets

This directory intentionally keeps a small set of distributable demo artifacts
that are referenced by `README.md` and `DEMO.md`.

Tracked content:

- `controlled_demos/`: the 4 curated controlled demo scenes
- `spontaneous_demos/`: the 4 curated spontaneous demo scenes
- `control_track_example/`: one full timing-track example with its audio and
  metadata

All other locally generated outputs should stay ignored by Git.
